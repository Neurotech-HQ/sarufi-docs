# What is Sarufi ?

At Sarufi, we're on a mission to empower African developers to build chatbots and conversational experiences that are tailored to African languages. We believe that language is at the core of communication, and our goal is to **democratize natural language processing tools** around African languages.

With our **lightweight** and **blazingly fast** platform, you can build chatbots and integrate them with messaging platforms like WhatsApp and Telegram, **without worrying about the complexities of natural language processing**. We handle all the **heavy lifting**, including **intent detection**, **entity recognition**, **natural language understanding**, and **intent fulfillment**, so that you can **focus on building your bot**.

Our vision is to **transform the way customers engage with businesses in Africa**, by making it easier to create innovative conversational experiences. Join us on our journey to build the **future of conversational AI** that truly represents the **diversity** of African languages.

![Sarufi](/img/undraw_artificial_intelligence_re_enpp.svg)

:::tip What does Sarufi mean ?
Sarufi is a Swahili word that means `grammar`.
:::

## Getting Started

To get started [Sarufi](https://sarufi.io/) you will need to create an account on [sarufi.io](https://sarufi.io/). Once you have created an account from [sarufi.io](https://sarufi.io/) you will get credentials to start building your chatbot.

### Supported Languages

- [x] Python 3.6+
  - [Sarufi Python SDK](https://github.com/sarufi-io/sarufi-python-sdk)
- [x] Nodejs 12+
  - [Sarufi Nodejs SDK](https://github.com/sarufi-io/sarufi-js)
- [x] Golang 1.15+
  - [Sarufi Golang SDK](https://github.com/sarufi-io/sarufi-golang-sdk)
- [x] PHP 7.4+
  - [Sarufi PHP SDK](https://github.com/sarufi-io/sarufi-php)
  - [Sarufi Laravel SDK](https://github.com/sarufi-io/laravel-sarufi)
- [x] Dart 2.12+
  - [Sarufi Dart SDK](https://github.com/sarufi-io/sarufi-dart-sdk)

:::tip Other Languages
We natively maintain only the [Python SDK](https://github.com/sarufi-io/sarufi-python-sdk) for now, all the other SDKs are community maintained.
If you are interested in maintaining any of the SDKs or creating a new one for another language,
please reach out to us on [Telegram](https://t.me/neurotechafrica).
:::

## Installation

As a developer using **Sarufi** conversational AI platform, you have two options for installing the required SDKs:

1. You can install language-based SDKs directly from the official repositories mentioned above.

2. Alternatively, you can install the official **Sarufi** SDKs from the official release through package managers.

This gives you the flexibility to choose the installation method that works best for your specific needs and development environment.

<Tabs>
<TabItem value="js" label="JavaScript">

```bash
$ npm install sarufi
```

</TabItem>

<TabItem value="py" label="Python">

```bash
$ pip install --upgrade sarufi
```

</TabItem>

<TabItem value="go" label="Golang">

```bash
$ go get github.com/sarufi-io/sarufi-golang-sdk
```

</TabItem>

<TabItem value="php-laravel" label="PHP">

```bash
$ composer require alphaolomi/sarufi-php
```

</TabItem>

<TabItem value="php" label="Laravel">

```bash
$ composer require alphaolomi/laravel-sarufi
$ php artisan vendor:publish --tag="sarufi-config"
```

</TabItem>

<TabItem value="dart" label="Dart">

```bash
$ flutter pub add sarufi
```

</TabItem>

</Tabs>

:::tip
If you don't see your language here, please go to official repositories and install the SDK from there or you can also reach out to us on [Telegram](https://t.me/neurotechafrica).
